initModel.ConvertToDlString(s)¶

The function converts strings commonly used in RSTAB / RFEM so that they can be used In WebServices. It solved issue #4. Examples: ‘1,3’ -> ‘1 3’ ‘1, 3’ -> ‘1 3’ ‘1-3’ -> ‘1 2 3’ ‘1,3,5-9’ -> ‘1 3 5 6 7 8 9’

Params:

RSTAB / RFEM common string

Returns a WS conform string.
